  <behavior>
    VAD:
    - VadDetector.is_speech(audio_block) returns True for speech audio, False for silence/noise
    - VadDetector processes 512-sample sub-chunks at 16kHz
    - VadDetector uses configurable threshold (default 0.5)
    - VadDetector runs on CPU (torch.device("cpu")), never GPU

    AudioCapture:
    - AudioCapture.start(device_index) opens sounddevice InputStream at 16kHz mono float32
    - AudioCapture.stop() closes the stream
    - Audio callback puts blocks into a thread-safe queue via put_nowait
    - Queue has maxsize=200 (~20 seconds buffer)
    - Queue full condition drops audio silently (no blocking)
    - list_input_devices() returns available microphone devices
    - find_device_by_name(name) returns device index by partial match

    Chunker:
    - AudioChunker accumulates audio blocks from the queue
    - When VAD detects silence gap >= 1.5 seconds, finalizes the chunk
    - When accumulated audio reaches 20 seconds, forces a chunk cut regardless of VAD
    - Each finalized chunk includes ~1 second overlap from the START of the next chunk
    - Noise-only blocks (VAD says no speech for the entire block) are skipped
    - get_chunk() returns a finalized numpy audio array or None if no chunk ready
  </behavior>
  <action>
    Implement the audio capture and processing pipeline.

    1. Create `src/dental_notes/audio/capture.py`:
       - Class `AudioCapture` with `__init__(self, settings: Settings)`
       - `audio_q: queue.Queue[np.ndarray]` with maxsize=200
       - `_audio_callback(indata, frames, time_info, status)` -- copies and flattens indata, puts into queue via put_nowait, silently drops on queue.Full
       - `start(device_index: int | None = None)` -- opens sd.InputStream(samplerate=16000, channels=1, dtype="float32", blocksize=1600, callback=self._audio_callback, device=device_index)
       - `stop()` -- closes and aborts the stream
       - `get_block() -> np.ndarray | None` -- gets from queue with timeout=0.1, returns None on Empty
       - Module-level `list_input_devices() -> list[dict]` and `find_device_by_name(name: str) -> int | None` functions (per research pattern)
       - `get_level() -> float` -- returns RMS of most recent audio block (for UI level indicator)

    2. Create `src/dental_notes/audio/vad.py`:
       - Class `VadDetector` with `__init__(self, threshold: float = 0.5)`
       - Loads silero-vad model on CPU: `torch.hub.load(repo_or_dir="snakers4/silero-vad", model="silero_vad", trust_repo=True)` but wrap in try/except and provide clear error if torch not available
       - `is_speech(audio_block: np.ndarray) -> bool` -- processes 512-sample sub-chunks, returns True if max probability > threshold
       - `get_speech_probability(audio_block: np.ndarray) -> float` -- returns the max speech probability for the block
       - Force CPU: all torch tensors created with `torch.from_numpy(...).to("cpu")`
       - `reset()` -- resets VAD model state (silero-vad is stateful between calls)

    3. Create `src/dental_notes/transcription/chunker.py`:
       - Class `AudioChunker` with `__init__(self, settings: Settings, vad: VadDetector)`
       - Internal buffer: list of numpy arrays
       - Tracks: total samples accumulated, samples since last speech, overlap buffer from previous chunk
       - `feed(audio_block: np.ndarray) -> np.ndarray | None`:
         - Runs VAD on the block
         - If no speech and buffer is empty: skip (return None) -- noise-only filtering
         - If no speech and buffer has content: increment silence counter
         - If speech: reset silence counter, append to buffer
         - If silence >= SILENCE_GAP_SECS * SAMPLE_RATE: finalize chunk, save overlap, return chunk
         - If total duration >= MAX_CHUNK_DURATION_SECS: force-finalize, save overlap, return chunk
         - Otherwise return None (still accumulating)
       - `flush() -> np.ndarray | None` -- finalizes whatever is in the buffer (called on session stop)
       - Overlap handling: when finalizing, save the last OVERLAP_SAMPLES (16000 = 1 second) as the start of the next chunk's buffer

    Implement the transcript stitching and writing modules.

    1. Create `src/dental_notes/transcription/stitcher.py`:
       - Function `deduplicate_overlap(prev_text: str, new_text: str, max_overlap_words: int = 10) -> str`
       - Split both texts into words
       - Compare tail of prev_text (up to max_overlap_words) with head of new_text
       - Find longest matching suffix of prev tail that equals prefix of new head
       - Return new_text with the overlapping prefix words removed
       - Handle edge cases: empty strings, no overlap found, single-word overlap
       - Per research: use simple word-level matching first (not fuzzy). Research acknowledges this may need fuzzy matching later but simple matching is the correct starting point.

       - `append(self, text: str) -> None`: write text, flush, os.fsync(fileno) -- crash safety
       - `close(self) -> None`: close file handle
       - `path` property: returns Path to the transcript file
       - Context manager support (__enter__/__exit__) for safe cleanup
