         d. Return "" (empty string -- chunk is skipped but audio is preserved)
         e. Keep self._transcription_behind = True (UI will show alert)
       - Do NOT use tenacity here -- the retry logic needs custom behavior (torch.cuda.empty_cache between retries, raw audio save on exhaustion) that is simpler as a manual loop than a tenacity callback chain.
